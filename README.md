# sauer-compass
 Compass hud for Cube 2: Sauerbraten which gives information about the direction the player is looking.
 set `gamehud` or `edithud` equal to `hud` after executing this script.
 ```
 exec sauer-compass/compass.cfg
 edithud = hud
 ```
